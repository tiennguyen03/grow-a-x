# PastSprints — archived sprint-section versions

When a **single sprint section** inside an epic doc (e.g. `## Sprint 1A …`) is
meaningfully revised, the previous version of just that section is copied here
*before* the edit.

**Naming:** `Epic<N>-Sprint<NX>.<YYYY-MM-DD>.md`
(e.g. `Epic1-Sprint1A.2026-06-18.md`).

Use this for sprint-scoped edits. If the change is bigger than one sprint
(epic title/status, overview tables, multiple sprints), archive the whole epic
file to [`../PastEpics/`](../PastEpics/) instead.

> This is a manual mirror for convenience — **git history is still the source of
> truth.** See "Documentation convention" in `CLAUDE.md`.
